# 🌌 OMC VR-OS | SENTINEL [ALPHA V6.0]

Bem-vindo ao **SENTINEL**, o módulo de blindagem sensorial e soberania operativa do ecossistema OMC. Esta versão Alpha foi liberada para coleta de telemetria e feedback de usuários pioneiros.

## 🛠 Instruções de Inicialização

Para garantir que o sistema opere com **Baixa Latência**, siga rigorosamente os protocolos abaixo:

### 1. Requisitos de Ambiente
* **Navegador:** Recomendamos Google Chrome ou Microsoft Edge (necessário para suporte à API de Reconhecimento de Voz).
* **Protocolo HTTPS:** O sistema só ativará o JARVIS (voz) e o WebXR (VR) se for acessado via link seguro (HTTPS).
* **Áudio:** Utilize fones de ouvido para uma experiência imersiva de feedback bioacústico.

### 2. Comandos Operativos (Teclado)
O Kérnel v6.0 utiliza o protocolo **Push-to-Talk** para evitar ruídos no barramento:
* **[ Tecla J ] (Pressionar e Segurar):** Ativa a escuta do JARVIS. Fale seu comando enquanto mantém a tecla pressionada.
* **[ Tecla F ] (Opcional):** Alterna o modo de Shadow (Foco Total).

### 3. Entendendo o HUD (Interface)
* **LATÊNCIA:** Localizado no canto superior. Se o contador ultrapassar **60s**, o sistema entrará em *Glitch Alert* (tremor visual) para forçar o reinício da execução.
* **MIELINA:** Indica sua velocidade de processamento neural estimada.
* **BATERIA CPF:** Representa a carga de energia do seu Córtex Pré-Frontal. Microdecisões e distrações drenam este recurso.

## 🧪 O que Testar?
1. **Interação com JARVIS:** Pressione 'J' e diga comandos como "Status do Sistema" ou "Ativar Shadow Mode".
2. **Navegação Espacial:** No desktop, use o mouse para olhar ao redor. Em VR, mova a cabeça para explorar o cockpit.
3. **Resiliência ao Glitch:** Observe como o sistema reage quando você fica inativo por muito tempo.

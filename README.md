# omc-vros-sentinel-alpha
Interface Experimental de Soberania Operativa e Blindagem Sensorial. Este repositório hospeda a versão Alpha do SENTINEL v6.0, focada na redução de latência cognitiva e monitoramento de biofeedback (Mielinização e Córtex Pré-Frontal). Desenvolvido para testes de UX em ambientes WebXR e Desktop.
